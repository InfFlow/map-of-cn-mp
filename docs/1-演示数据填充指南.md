# Map of Us · 演示数据填充指南

> 目的：让相册、地图、热力图、年历、标签云、数据大屏都「填满」，demo 才有说服力。
> 建议录入 **15-20 段足迹**，覆盖 **8-10 个省**、**3 个以上年份**、四季都有，每段配 **3-5 张照片**。

---

## 一、最快的录入方式（小程序内，推荐）

### 路径 A：时间线「＋ 记录」（最快，适合批量补历史）
1. 底部 Tab →「时间线」
2. 右下角 **＋ 记录** 浮动按钮
3. 填：省份 → 城市（二级联动）→ 日期 → 标题 → 手记 → 选 3-5 张照片 → 保存
4. 保存时会自动用城市名查经纬度，**自动落到地图上**

> 这是补历史数据最快的入口，一段约 30 秒。

### 路径 B：足迹地图「＋ 打卡」（适合"此刻在这里"）
1. 底部 Tab →「行程」邻近的足迹地图，或「我的 → 回顾与统计 → 足迹地图」
2. 右下角 **＋ 打卡** → 自动定位 → 确认省市 → 填地点名 → 传照片 → 打卡

## 三、必配的「纪念日」（首页/时间线顶部要用）

在「我的 → 编辑我们的回忆 → 纪念日」里加：

1. **在一起 100 天**（关键！）：日期填在一起满 100 天那天，标题写「第 100 天」
   - 系统靠「第 N 天」这个格式**反推在一起的起点**，首页"在一起第 X 天"、数字滚动都依赖它
2. **周年纪念**：勾选「每年重复」，首页会显示"X 天后 · 周年纪念"倒计时
3. 生日、第一次见面等，按需加

## 四、照片建议

- 每段足迹 **3-5 张**，相册、详情页、回顾放映、海报都会用到
- 横图竖图都行，系统统一 `aspectFill` 裁切
- 第一张会作为**封面**（时间线卡片、足迹卡都取第一张）
- 有照片的足迹，AI 才能生成手记、精华筛选才有素材
